Какие бывают CSS селекторы

CSS селекторы можно разбить на 4 категорий

  • Простые
  • Комбинированные
  • Псевдо
  • Атрибутные

Простые

Селекторы элемента
В качестве селектора может выступать любой элемент HTML, например селектор тега <table> будет применен ко всем <table> элементам.
id селекторы
Этот селектор описывает элемент к которму назначен атрибут id. В докемунте может присутствовать лишь один элемент с заданным идентификатором, например #logo и он будет применем лишь к одному элементу с заданным id.
Селекторы по классам
В данном случае селектором является class элемента который также является его атрибутом, например .logo и селектор применится ко всем эелементам с классом .logo.
Универсальный селектор
Такой селектор может потребоваться когда нам нужно применить одинь стиль для всех элементов веб страницы. Он обозначается звездочкой * и под ней может подразумеваться любой элемент разметки.
Селектор группировки
Данный селектор выбирает все элементы HTML с одинаковыми определениями стиля. В таком случае будет лучше сгрупировать селекторы чтобы минимизировать код. Например, элементы h1 h2 h3 будут записаны в одну строку через запятую и уже потом к ним применяются одинаковые свойства.
