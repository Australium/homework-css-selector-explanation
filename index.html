Какие бывают CSS селекторы

CSS селекторы можно разбить на 4 категорий

  • Простые
  • Комбинированные
  • Псевдо
  • Атрибутные

Простые

Селекторы элемента
В качестве селектора может выступать любой элемент HTML, например селектор тега <table> будет применен ко всем <table> элементам.
id селекторы
Этот селектор описывает элемент к которму назначен атрибут id. В докемунте может присутствовать лишь один элемент с заданным идентификатором, например #logo и он будет применем лишь к одному элементу с заданным id.
Селекторы по классам
В данном случае селектором является class элемента который также является его атрибутом, например .logo и селектор применится ко всем эелементам с классом .logo.
Универсальный селектор
Такой селектор может потребоваться когда нам нужно применить одинь стиль для всех элементов веб страницы. Он обозначается звездочкой * и под ней может подразумеваться любой элемент разметки.
Селектор группировки
Данный селектор выбирает все элементы HTML с одинаковыми определениями стиля. В таком случае будет лучше сгрупировать селекторы чтобы минимизировать код. Например, элементы h1 h2 h3 будут записаны в одну строку через запятую и уже потом к ним применяются одинаковые свойства.

Комбинированные

Вложенные селекторы
Стилизация будет применена к элементу который вложен в другой элемент, например, в контейнере <div> может находится вложенный элемент <p>, и если мы хотим применить к этому элементу стиль, то нам как раз поможет в этом вложенный селектор. Таким образом можно также применить стили к обоим элементам.
Дочерние селекторы
Дочерним называется элемент, который непосредственно располагается внутри родительского элемента. Именно к нему и будут применены стили. Например, все дочерние элементы <dt> будут выбраны когда они вложены на первом уровне по отношению к элементу <dl>.
Родственные селекторы
Данные селекторы применяют стили ко всем близлежащим элементам. К примеру, для селектора h1~p стиль будет применяться ко всем элементам <p>, располагающихся после заголовка <h1>. При этом <h1> и <p> должны иметь общего родителя.
Соседние селекторы
Этот вариант включает в себя символ плюса (+), который устанавливается между двумя заданными селекторами. Стили будут применены к элементу который находится непосредственно после указанного элемента, если у них общий родитель. Например, селектор ul + li выберет любой <li> элемент, который находится непосредственно после элемента <ul>.

Псевдо

Псевдо-классовые
Псевдокласс в CSS — это ключевое слово, добавленное к селектору, которое определяет его особое состояние. Например, :hover может быть использован для изменения цвета кнопки при наведении курсора на неё.
Псевдо-элементные
Псевдоэлемент в CSS — это ключевое слово, добавляемое к селектору, которое позволяет стилизовать определённую часть выбранного элемента. Например, псевдоэлемент ::first-line может быть использован для изменения шрифта первой строки абзаца.

Атрибутные

[атрибут]
Устанавливает стиль для элемента, если задан специфичный атрибут. Его значение в данном случае не важно. Например, в селекторе a[target] будут применены стили к элементам <a> с атрибутом target.
[атрибут="значение"]
Устанавливает стиль для элемента в том случае, если задано определённое значение специфичного атрибута. Например, в селекторе a[target=_blank] будут применены стили к элементам <a> с атрибутом target=_blank.
[атрибут^="значение"]
Устанавливает стиль для элемента в том случае, если значение атрибута начинается с указанного текста. Например, в селекторе [class^="top"] выбираются все элементы атрибута class, чье значение начинается с текста "top".
[атрибут$="значение"]
Устанавливает стиль для элемента в том случае, если значение атрибута оканчивается указанным текстом. Например, в селекторе [class^="goodbye"] выбираются все элементы атрибута class, чье значение заканчивается текстом "goodbye".
[атрибут*="значение"]
Устанавливает стиль для элемента в том случае, если значение атрибута имеет указанный текст. Например, в селекторе [href*="htmlbook"] выбираются все элементы атрибута href, чье значение имеет текст "htmlbook".
[атрибут~="значение"]
Устанавливает стиль для элемента в том случае, если значение атрибута имеет указанное слово. Например, в селекторе [title~="flower"] выбираются все элементы атрибута title, чье значение имеет слово "flower".
[атрибут|="значение"]
Устанавливает стиль для элемента в том случае, если значение атрибута точно равно указанному значению или начинается с указанного значения, после которого идёт дефис. Например, в селекторе [class|=top] выбираются все элементы атрибута class, чье значение начинается с "top".
